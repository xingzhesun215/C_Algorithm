2.2.1数组
数组结构就是一排紧密相连的可数内存,并提供了一个能够直接访问单一数据内容的计算方法
数组 int score[5]
二维数组 int score[行大小][列大小]

2.2.2链表
链表是由许多相同数据类型的数据项按照特定顺序排列而成的线性表, 链表的特性是各个数据项在计算机内存中的位置是不连续且随机存放的,优点是数据的插入和删除特别方便,缺点是设计麻烦,并且查找数据需按序查找.
单向列表 类似火车,一个节点有数据字段和指针2个元素组成,指针指向下一个元素在内存中的地址.

2.2.3堆栈
堆栈是一群相同数据形态的组合,所有的动作均在顶端进行,具有后进先出的特性
堆栈的基本运算
create
push
pop
empty
full

2.2.4队列
队列是一群相同数据形态的组合,所有的加入和删除动作都发生在不同的两端,并且符合先进先出的特性.
队列基本运算
create
add
delete
front
empty
